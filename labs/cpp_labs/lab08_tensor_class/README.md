# Lab 08: Tensor Class - Building a Mini PyTorch-like Tensor

## Overview
Combine all previous concepts to build a complete tensor class similar to PyTorch's C++ backend. This integrates memory management, move semantics, templates, and zero-copy views.

## Learning Objectives
1. Design a production-quality tensor class
2. Implement efficient operations (add, mul, matmul)
3. Support multiple dtypes using templates
4. Add broadcasting and reshaping
5. Integrate all previous lab concepts

## Estimated Time
2-3 hours

## Prerequisites
- All previous labs (01-07)

## Connection to vLLM
- Understanding PyTorch's ATen tensor implementation
- Building custom operators for vLLM
- Efficient tensor manipulation in C++
