# AIBrix

[AIBrix](https://github.com/vllm-project/aibrix) is a cloud-native control plane for vLLM that simplifies deployment, scaling, routing, and LoRA adapter management for large language model inference on Kubernetes. It provides enterprise-scale features such as smart routing, LLM-specific autoscaling, distributed KV cache, and high-density LoRA management.

For installation and usage instructions, please refer to the [AIBrix documentation](https://github.com/vllm-project/aibrix).
